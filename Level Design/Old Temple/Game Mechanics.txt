Game Mechanics
	In this section, I will only list some important game mechanics. If you want to see all the game mechanics that I used, you can download the level.

Staff
	The most important mechanic I added to this level is the Staff. The Staff can trigger the Conveying hanger and combat with the enemy.

Enemy
	I set few enemies patrolling by given routes.
	Players can choose to combat with these enemies by using the staff or dodge from them stealthy.
